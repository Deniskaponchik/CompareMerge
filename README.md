## MergeSort
### Для запуска
1. Скачать на компьютер папку RunMe
2. Должна быть установлена Java на ПК и прописана переменная
3.

### Аргументы
**java -jar**  Предварительные обязательные аргументы
**MergeSort.jar**  Имя запускаемой программы
**-a**
